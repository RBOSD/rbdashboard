## 三、取得資料（Power Query）

### 步驟 2：連接 Excel

1. 開啟 Power BI Desktop
2. 點選 **首頁** → **取得資料** → **Excel**
3. 瀏覽到 `C:\Users\hckuo\Desktop\Power BI\tra_test_data.xlsx`
4. 選擇 **工作表1**，點選 **轉換資料**（進入 Power Query 編輯器）

### 步驟 3：在 Power Query 中整理資料

1. **新增「年月」欄位**（供趨勢圖使用）：
   - 選取「發生時間」欄 → **新增欄** → **日期** → **年** → 命名為 `年`
   - 選取「發生時間」欄 → **新增欄** → **日期** → **月** → 命名為 `月`
   - **新增欄** → **自訂欄** → 欄位名稱：`年月`，自訂欄公式：
     ```
     Text.From(Date.Year([發生時間])) & "/" & Text.PadStart(Text.From(Date.Month([發生時間])), 2, "0")
     ```
   - 若需**民國年**（如 114/08）：將 `Date.Year([發生時間])` 改為 `Date.Year([發生時間]) - 1911`

2. **統一營運機構格式**（可選）：若 TRA 要顯示為「臺鐵」：
   - **新增欄** → **條件式欄位** → 欄位名稱：`營運機構名稱`
   - 條件：若 `[營運機構] = "TRA"` 則 `"臺鐵"`，若 `"THSR"` 則 `"高鐵"`，若 `"AFR"` 則 `"林鐵"`，若 `"TSR"` 則 `"糖鐵"`，否則 `[營運機構]`

3. **檢查資料型別**：
   - 發生時間：日期/時間
   - 營運機構、事故事件分類、事故事件種類、原因第一階～第三階：文字

4. 點選 **關閉並套用**

## 四、建立資料模型

### 步驟 4：資料檢視

1. 左側點選 **資料檢視**（圖示資料表）
2. 確認 `資料表1`（或您重新命名）有正確欄位
3. 新增 **量值**（用於計數）：

   - 在 **資料表1** 上按右鍵 → **新增量值**
   - 量值名稱：`件數`
   - DAX 公式：`件數 = COUNTROWS('資料表1')`
   - 或使用：`件數 = COUNT('資料表1'[項次])`

## 五、建立視覺效果

### 步驟 5：頂部指標卡（4 張）

1. **首頁** → **卡片**
2. 將「卡片」拖到畫布，調整大小
3. 在 **視覺效果** 的 **欄位** 中：
   - **值**：拖入 `件數`（或建立篩選後件數的量值）
4. 複製 4 張卡片，分別設定：
   - 卡片 1：總件數（篩選：全部）
   - 卡片 2：重大行車事故（篩選：事故事件分類 = 重大行車事故）
   - 卡片 3：一般行車事故（篩選：事故事件分類 = 一般行車事故）
   - 卡片 4：異常事件（篩選：事故事件分類 = 異常事件）

5. **篩選方式**：在每張卡片上按右鍵 → **新增篩選器** → 選擇「事故事件分類」，或使用 **篩選器** 窗格設定。

6. **建議**：建立 4 個量值，讓卡片自動依篩選連動：
   ```
   總件數 = COUNTROWS('資料表1')
   重大件數 = CALCULATE('總件數', '資料表1'[事故事件分類] = "重大行車事故")
   一般件數 = CALCULATE('總件數', '資料表1'[事故事件分類] = "一般行車事故")
   異常件數 = CALCULATE('總件數', '資料表1'[事故事件分類] = "異常事件")
   ```
   然後將這 4 個量值分別放入 4 張卡片。

2. **圖例**：`營運機構`
3. **值**：`件數`（或 總件數）
4. **互動**：點選圓餅可篩選其他圖表（預設交叉篩選）

### 步驟 8：事件分類佔比（圓餅/環圈圖）

2. **圖例**：`事故事件分類`
3. **值**：`件數`

### 步驟 9：肇因深度鑽研（橫條圖）

1. **視覺效果** → **橫條圖**
2. **Y 軸**：`原因第一階`（或第二階、第三階）
3. **X 軸**：`件數`
4. **鑽研**：
   - 在 **資料** 窗格中，將 `原因第一階`、`原因第二階`、`原因第三階` 依序拖入 **鑽研** 或 **階層**
   - 或使用 **鑽研向下** 按鈕：點選圖表 → 右鍵 → 鑽研 → 鑽研向下

5. **建立鑽研階層**：
   - 在 **資料** 窗格中，將 `原因第一階`、`原因第二階`、`原因第三階` 依序拖到同一個圖表的 **Y 軸**，會形成階層
   - 點選圖表上的長條可向下鑽研

### 步驟 10：熱區矩陣（營運機構 vs 肇因）

**方式 A：Power BI 內建矩陣（⭐ 互動版必用）**

1. **視覺效果** → **矩陣**
2. **列**：將 `原因第一階`、`原因第二階`、`原因第三階` 依序拖入（形成鑽研階層）
3. **欄**：`營運機構名稱`（或 營運機構）
4. **值**：`件數`
5. **條件式格式（紅色漸層）**：
   - 選取矩陣 → **格式** → **儲存格項目** → **背景色彩**
   - 選擇 **規則** → 依據「值」、摘要「總和」
   - 最小值：白色 `#FFFFFF`，最大值：紅色 `#E02F44`
   - 或自訂：最小 `#181B1F`、最大 `#E02F44`（暗黑風格）
6. **互動**：點選其他圖表時，此矩陣會自動篩選並更新

**何時用 Python、何時用矩陣**：

| 需求 | 建議 |
|------|------|
| 要與其他圖表互動（點擊篩選） | **Power BI 矩陣** + 條件式格式 |
| 要精確還原網頁表格樣式 | Python（但無法互動） |

**方式 B：使用 Python 視覺效果（靜態，無法與其他圖表互動）**

1. **視覺效果** → 點選 **取得更多視覺效果** → 搜尋 **Python** 或 **Py**
2. 若無安裝：點選 **從 AppSource 取得**，安裝 **Python 視覺效果**
3. 新增 Python 視覺效果後，將以下欄位**全部**拖入「值」：
   - `原因第一階`、`原因第二階`、`原因第三階`（三階都拖入才能連動鑽研）
   - `營運機構名稱`
   - `件數`（或 項次）
4. 在程式碼編輯器中，貼上 `powerbi_python_heatmap.py` 的內容
5. 點選 **執行** 產生熱區圖
6. **連動設定**：確保此 Python 視覺效果參與交叉篩選（預設為是）。當您點選其他圖表（如肇因鑽研橫條圖、圓餅圖）時，熱區圖會自動依篩選結果切換顯示一階／二階／三階。

> 注意：需在 Power BI Desktop 的 **檔案** → **選項** → **Python 指令碼** 中設定 Python 安裝路徑，並安裝 `pandas`、`matplotlib`。

**若出現「too many values to unpack」錯誤**：請改用精簡版 `powerbi_python_heatmap_simple.py`，且**只拖入 3 個欄位**（原因第一階、營運機構名稱、件數），依序放在「值」的前三個位置。

**方式 C：完整看板（單一 Python 視覺效果呈現整個畫面）**

1. 新增 Python 視覺效果
2. 將以下欄位**全部**拖入「值」：
   - `發生時間`、`營運機構`、`事故事件分類`、`原因第一階`、`原因第二階`、`原因第三階`、`件數`（或 項次）
3. 貼上 `powerbi_dashboard_full.py` 的完整程式碼
4. 執行後會顯示：頂部 4 張指標卡、趨勢圖、營運機構圓餅、事件分類圓餅、肇因橫條圖、熱區矩陣表格，全部在一個畫面中。

## 九、Power BI + Python 環境設定

若要使用 Python 視覺效果（熱區圖）：

1. **安裝 Python**：建議 Python 3.8 以上，並勾選「Add to PATH」
2. **安裝套件**：在命令提示字元執行
   ```
   pip install pandas matplotlib
   ```
3. **Power BI 設定**：**檔案** → **選項與設定** → **選項** → **Python 指令碼**
   - 設定 **偵測到的 Python 主目錄** 或手動指定（例如 `C:\Python311\python.exe`）

## 十、常見問題

**Q：發生時間顯示為數字？**  
A：在 Power Query 中將該欄位轉換為「日期/時間」型別。

**Q：營運機構 TRA 要顯示「臺鐵」？**  
A：在 Power Query 新增「取代值」或「條件式欄位」。

**Q：如何讓鑽研有「返回上一層」？**  
A：在肇因圖表上按右鍵 → **鑽研** → **鑽研向上**，或點選圖表左上角箭頭。

**Q：熱區矩陣顏色如何設定？**  
A：選取矩陣 → 格式 → 儲存格項目 → 背景 → 規則 → 設定最小/最大色彩與值。
